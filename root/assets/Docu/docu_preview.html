PREVIEW

The Preview

In the Preview you can see the generated RITS, as you defined it before in the sections. The preview is editable and can be changed individually, as needed.

The Button Copy will select the RITS and you can paste it into Salesforce with Ctrl + v

The Button Clear will clear all the data on the screen and you can start a new RITS
